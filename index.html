// What Is ORCAI //

orcai is a Go BubbleTea TUI that turns your terminal into an AI command center. Manage tmux sessions from a BBS-style sidebar, wire up any CLI tool as a plugin via gRPC or sidecar YAML, and run multi-step AI agent pipelines across Claude, Ollama, GitHub Copilot тАФ all from one dark, dense, beautiful interface.

тФМтФА TMUX SESSIONS тФАтФР

Manage AI agent workspaces from a BBS-style sidebar TUI. Spawn, switch, and kill tmux sessions without leaving your workflow.

тФМтФА PLUGIN SYSTEM тФАтФР

gRPC plugins + sidecar YAML. Any CLI tool becomes a pipeline node. Tier 1 native Go. Tier 2 wrappers for everything else.

тФМтФА AGENT PIPELINES тФАтФР

YAML pipelines run Claude, Ollama, and Copilot at scale. Chain tools with typed inputs and outputs.
